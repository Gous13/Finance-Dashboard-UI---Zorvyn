# 💰 Finance Dashboard UI

A clean and interactive finance dashboard built using React.js and Tailwind CSS. This project focuses on frontend design, component structure, and state management to simulate a real-world financial tracking interface.

## 🚀 Features

- 📊 Dashboard Overview with summary cards (Balance, Income, Expenses)
- 📈 Time-based visualization (balance trend)
- 🥧 Categorical visualization (spending breakdown)
- 📋 Transactions table with:
  - Search
  - Filtering
  - Sorting
- 🔐 Role-based UI (Viewer / Admin)
- 💡 Insights section (spending analysis, monthly comparison)
- ⚛️ State management using Context API
- 📱 Fully responsive design
- 🌙 Optional dark mode support

## 🛠️ Tech Stack

- React.js (JavaScript)
- Tailwind CSS
- Recharts (for data visualization)
- Context API (state management)

## 📂 Project Structure

- Reusable components for cards, charts, tables, and filters
- Centralized state management
- Mock data used (no backend required)

## 🎯 Objective

This project was developed as part of a frontend assessment to demonstrate UI/UX design, state handling, and component-based architecture without relying on backend services.
